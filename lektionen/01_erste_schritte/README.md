# Lektion 01 · Erste Schritte – LED

In dieser Lektion lernt ihr, wie man mit wenigen Python-Zeilen eine LED
steuern kann – zuerst die **interne** LED auf dem Pico, dann eine **externe**
LED auf dem Breadboard.

## Lernziele

- Ihr versteht, was ein **GPIO-Pin** ist.
- Ihr könnt eine LED ein- und ausschalten.
- Ihr nutzt `while True`, `sleep` und Variablen.

## Python-Konzepte

- `import` – Module einbinden
- Variablen
- `while True` – Endlosschleife
- `sleep(sekunden)` – warten
