# Lektion 02 · Eingaben – Taster und Potentiometer

Mikrocontroller reagieren auf die Außenwelt. In dieser Lektion lernt ihr zwei
grundlegende Eingaben kennen:

1. **Digitale Eingänge** – ein Taster ist an (gedrückt) oder aus.
2. **Analoge Eingänge (ADC)** – ein Potentiometer liefert Werte in einem Bereich.

## Lernziele

- `Pin.IN` mit `Pull-Up` verstehen und verdrahten
- `if / else`-Abfragen benutzen
- Den **Analog-Digital-Wandler** (ADC) einsetzen

## Dateien

| Datei | Thema |
|-------|-------|
| `01_taster_lesen.py` | Tastendruck erkennen und im Terminal ausgeben |
| `02_led_per_taster.py` | LED leuchtet, solange Taster gedrückt ist |
| `03_potentiometer.py` | Analogwert eines Potis auslesen |
