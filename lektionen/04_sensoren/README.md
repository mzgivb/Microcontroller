# Lektion 04 · Sensoren

Sensoren wandeln Umweltgrößen (Licht, Temperatur, Feuchtigkeit) in elektrische
Signale um. In dieser Lektion baust du die **Messbausteine** für die folgenden
Projekte.

## Lernziele

- Den Unterschied zwischen **analogen** und **digitalen** Sensoren
- **Schwellenwerte** mit `if/else` abbilden
- Werte sinnvoll skalieren und anzeigen
